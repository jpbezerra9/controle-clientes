Sistema de Gestão de Clientes

Aplicação full stack desenvolvida como projeto acadêmico, utilizando React no frontend e Laravel (API REST) no backend.

O sistema permite o gerenciamento completo de clientes, com funcionalidades de cadastro, edição, listagem e exclusão.

🛠️ Tecnologias Utilizadas

Frontend
React
Vite
Axios
Bootstrap
Backend
Laravel
MySQL
XAMPP

📌 Funcionalidades

✔ Cadastro de clientes
✔ Listagem de clientes
✔ Edição de dados
✔ Exclusão de clientes
✔ Validação de formulário
✔ Formatação automática (CPF/CNPJ e telefone)
✔ Interface responsiva
📷 Demonstração

⚙️ Como executar o projeto

🔵 Backend (Laravel)

cd C:\dev\Controle-clientes\backend\api-clientes
C:\xampp\php\php.exe artisan serve


🟢 Frontend (React)

cd C:\dev\Controle-clientes\frontend\clientes-react
npm install
npm run dev


🌐 Acessar

Frontend:

http://localhost:5173

Backend:

http://127.0.0.1:8000/api/clientes


🔒 Segurança

Os dados não são expostos na URL, pois a comunicação é feita via requisições HTTP (API REST).
A proteção real é realizada no backend, onde podem ser implementadas validações, autenticação e controle de acesso.

📚 Objetivo do Projeto

Desenvolver uma aplicação cliente-servidor com interface gráfica, aplicando conceitos de:

Arquitetura Cliente-Servidor
Modelo TCP/IP
Comunicação via HTTP
Integração entre frontend e backend
